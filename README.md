Kubernetes Demo App – Profile Manager

A simple full-stack demo application designed to showcase how to deploy and manage a multi-tier application on Kubernetes.

The app allows users to submit basic profile information (name, email, and interests) through a web interface, which is then stored in a PostgreSQL database.

This project demonstrates core Kubernetes concepts including:

Deploying a stateless application (web app)
Managing a stateful service (PostgreSQL)
Service-to-service communication inside the cluster
Environment variable configuration
Persistent storage for database data
Containerized application deployment using Docker

The goal of this project is to provide a hands-on, beginner-friendly example of how real-world applications are structured and deployed in Kubernetes.
